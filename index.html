Project Overview

This project implements a Convolutional Neural Network (CNN) to automatically classify satellite images into 10 different land use categories. Using the EuroSAT dataset containing 27,000 labeled satellite images, the model learns to distinguish between forests, highways, residential areas, crops, and other land types with exceptional accuracy.

Methodology

Dataset & Preprocessing

  • Dataset: EuroSAT RGB satellite images (64x64 pixels)
  • Data Split: 70% training (18,900) / 15% validation (4,050) / 15% test (4,050)
  • Normalization: Pixel values scaled from 0-255 to 0-1
  • Data Augmentation: Random horizontal/vertical flips and 90¬∞ rotations applied to training data
  • Batch Size: 32 images per batch

Training Configuration

  • Optimizer: Adam (adaptive learning rate)
  • Loss Function: Sparse Categorical Crossentropy
  • Callbacks: Early Stopping, ReduceLROnPlateau, ModelCheckpoint
  • Training Time: 50 epochs (~10 minutes)
  • Best Model: Saved from epoch 47

Key Learnings & Insights

  • Data Augmentation Impact: Random transformations significantly improved generalization by simulating real-world variations
  • Batch Normalization Benefits: Stabilized training and allowed for faster convergence
  • Callback Effectiveness: Automated learning rate reduction prevented plateaus and improved final accuracy
  • Architecture Design: Progressive filter increases (32‚Üí64‚Üí128‚Üí256) effectively captured hierarchical features
  • Class Imbalance Awareness: Similar vegetation categories required careful analysis to understand model behavior
